API、AIの出力、ソフトウェアは、今も多くの場面で盲目的に信頼されています。
REMEDAは、暗号学的証明によって「本当に信頼できるか」を検証します。

APIを試す APIキーを取得する

なぜ今必要か

AIの出力、APIレスポンス、ソフトウェア成果物は、簡単に改ざん・偽装・再利用されます。
検証できないものは、信頼できません。

問題

多くのシステムでは、完全性・実行証明・発行者証明・時間証明を同時に確認できません。

解決

REMEDAは、次の証拠を組み合わせて検証します。

  • 完全性：SHA256 による改ざん検知
  • 実行証明：CI / GitHub Actions の証拠
  • 発行者証明：GPG / Sigstore
  • 時間証明：Bitcoin / OpenTimestamps

Proで何が変わるか

Freeは基本検証。Proでは APIキー、GPG / Sigstore、検証履歴、高信頼スコアを使い、 チームや顧客に提出できる証明へ拡張できます。
